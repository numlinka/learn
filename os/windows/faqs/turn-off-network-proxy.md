# 关闭网络代理

## Windows 10

打开系统设置；点击 **网络和 Internet** 选项；

![](/static/image/f37ffa45.png)

点击 **代理** 找到手动设置代理，并关闭 **使用代理服务器** 选项；

![](/static/image/94faea8a.png)


## Windows 11

打开系统设置；点击 **网络和 Internet** 再点击 **代理** 选项；

![](/static/image/a0b4ce4a.png)

并关闭 **使用代理服务器** 选项；

![](/static/image/b85d16c3.png)
